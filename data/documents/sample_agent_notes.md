# Sample Notes About AI Agents

An AI agent is a system that can reason about a task, retrieve context, use tools, and produce an answer or action.

Information Retrieval is important for agents because language models have limited context windows and may not know current or private information.

A useful agent can retrieve information from memory, documents, web search, databases, or APIs.

Context ranking helps the agent choose the most relevant information instead of putting everything into the prompt.

Compaction means summarizing older information so it can still be useful without using too many tokens.
